# 🏎️ Garagem do Coder: Kotlin

Bem-vindo à Garagem do Coder! Este é um projeto desenvolvido em Kotlin que reúne uma série de algoritmos, cálculos matemáticos e simuladores de sistemas reais, tudo organizado em uma interface interativa via terminal.
O projeto começou com cálculos geométricos simples e evoluiu para uma estrutura complexa de menus, incluindo um simulador de caixa eletrônico.

🚀 Funcionalidades

O sistema é dividido em "Fases" de aprendizado (Hora de Codar), acessíveis através de um painel central:

📐 Hora de Codar 1: Geometria
Cálculo de área para diversas formas:
* Retângulo, Quadrado, Losango, Trapézio, Paralelogramo, Triângulo e Círculo.

🧠 Hora de Codar 2: Lógica e Decisão
* Verificação de números maiores, positivos/negativos.
* Cálculo de médias aritméticas com validações.
* Verificação de idade para votação.
* Calculadora de Peso Ideal.
* Calculadora básica de operações.

🔄 Hora de Codar 3: Laços de Repetição
* Contagem regressiva (KABOOM!).
* Algoritmos de divisão com validação (não aceita zero).
* Médias de notas de alunos com opção de repetir o processo.
* Tabuadas dinâmicas.

📦 Hora de Codar 4: Coleções (Lists)
* Cadastro e listagem de estudantes.
* Verificação de itens no Sistema Solar.
* Gerenciamento de lista de compras (Mercado).

🏦 Hora de Codar 5: Banco Farofa
Um simulador de Caixa Eletrônico completo com:
* Verificação de saldo e extrato.
* Depósitos, saques e transferências.
* Sistema de segurança por senha fixa.
